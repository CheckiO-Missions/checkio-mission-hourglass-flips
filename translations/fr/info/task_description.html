Un sablier est donné sous la forme d'un tableau {% if is_js %} [upper, lower] {% else %} tuple (upper, lower){% endif %} pour le nombre de minutes de sable qui sont actuellement stockées dans ses chambres supérieure et inférieure, les deux chambres étant suffisamment grandes pour contenir tout le sable de ce sablier. Ainsi, après que m minutes se soient écoulées, l'état de ce sablier particulier sera upper - min(upper, m), lower + min(upper, m). La quantité totale de sable à l'intérieur du sablier ne change jamais, et aucune des chambres ne contient d'anti-sable négatif.

Étant donné une liste de glasses, votre tâche consiste à trouver une séquence optimale de mouvements pour mesurer exactement t minutes, notées comme le nombre de retournements individuels du sablier effectués en cours de route. Chaque mouvement se compose de deux étapes:

  • vous devez d'abord attendre que le sablier qui contient actuellement la plus petite quantité non nulle m de sable dans sa chambre supérieure s'épuise.
  • lorsque cela se produit, choisissez un sous-ensemble quelconque de glasses et retournez instantanément ce sous-ensemble choisi.

Les cas de base de la récursion sont lorsque t est égal à 0 ou lorsqu'il reste exactement t minutes dans une chambre haute (aucun retournement n'est nécessaire), ou lorsque t est plus petit que le plus petit temps restant dans la chambre haute d'un sablier (aucune solution n'est possible). Sinon, mettez tous les sabliers dans leur nouvel état après m minutes, et parcourez en boucle tous les sous-ensembles possibles de glasses à retourner. Pour chacun de ces sous-ensembles, construisez récursivement la séquence optimale de mouvements pour mesurer les minutes restantes t - m, et combinez ces mouvements pour obtenir la meilleure solution.
